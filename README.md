# MacePvP Practice Map


## A MacePvP practice map with a powerful bot which can fight the player and be controlled to practice many techniques. Currently contains features such as:
- Stun slamming
- Ground fight and stun slamming evaluation and learning to improve the bot's playstyle when fighting you
- Dive bombing
- Qucik shield cooldown and a constant shield holding option (for stun slamming practice)
- and more!
### Each of which are toggleable in the map.
I also may be adding more features in the future.

# Requires herobot mod: https://modrinth.com/mod/herobot
